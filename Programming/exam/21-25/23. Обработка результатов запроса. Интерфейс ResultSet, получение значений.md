# 23. Обработка результатов запроса. Интерфейс `ResultSet`, получение значений

## 💡 Общая идея

`ResultSet` — это объект, который содержит результат SQL-запроса (`SELECT`).

Он представляет собой **таблицу данных**, которую можно читать построчно.

---

# 🟢 Как работает ResultSet

После выполнения запроса курсор находится **перед первой строкой**.

Чтобы начать чтение, нужно вызвать:

```java
rs.next()
```

---

## 🔁 Перемещение по данным

```text
before first row
↓ next()
row 1
↓ next()
row 2
↓ next()
...
↓ false (конец)
```

---

## 📌 Пример обхода

```java
ResultSet rs = ps.executeQuery();

while (rs.next()) {
    System.out.println(rs.getString("name"));
}
```

---

# 🔵 Получение данных

## Способы доступа к колонкам:

### 1. По имени столбца

```java
int id = rs.getInt("id");
String name = rs.getString("name");
```

---

### 2. По индексу столбца

```java
String name = rs.getString(2);
```

⚠ индексация начинается с 1

---

# 🟡 Основные методы ResultSet

## 📌 Перемещение

- `next()` — перейти к следующей строке
- `previous()` — назад (если поддерживается)
- `first()` — первая строка
- `last()` — последняя строка

---

## 📌 Получение значений

```java
getInt()
getString()
getDouble()
getBoolean()
getObject()
```

---

## 📌 Работа с NULL

```java
int value = rs.getInt("age");

if (rs.wasNull()) {
    // значение действительно NULL
}
```

---

# ⚠ Важные особенности

## 1. ResultSet — потоковый

```text
данные читаются последовательно
```

→ нельзя просто “прыгать” без специального типа курсора

---

## 2. Курсор перед первой строкой

Перед чтением обязательно:

```java
rs.next()
```

---

## 3. Закрытие ResultSet

Обычно закрывается автоматически вместе с `Statement`, но лучше использовать:

```java
try (ResultSet rs = ps.executeQuery()) {
    ...
}
```

---

# 💥 Частые ошибки

### ❌ забыть next()

```java
rs.getString("name"); // ошибка
```

---

### ❌ путаница NULL

```java
getInt() → может вернуть 0, но это не NULL
```

---

### ❌ неправильный индекс столбца

```text
индексация начинается с 1, а не с 0
```

---

# 🔗 Типичный шаблон работы

```java
ResultSet rs = ps.executeQuery();

while (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");

    System.out.println(id + " " + name);
}
```

---

# 🧠 Шпаргалка

* `ResultSet` — результат SELECT
* чтение всегда через `next()`
* данные берутся по имени или индексу
* индексы колонок начинаются с 1
* `wasNull()` нужен для проверки NULL
* ResultSet читается по строкам
