# 34. Пакет java.time. Классы для представления даты и времени

Пакет `java.time` — современный API для работы с датой и временем. Он пришёл на смену старым классам `Date` и `Calendar`.

---

## Почему Date и Calendar устарели

- `java.util.Date` — изменяемый (mutable), небезопасен в многопоточности.
- Месяцы в `Calendar` нумеруются с нуля (`JANUARY == 0`), что приводит к ошибкам.
- Нет разделения между датой, временем и часовым поясом.
- `java.time` (JSR-310, появился в Java 8) решает все эти проблемы.

---

## Основные классы

- `LocalDate` — дата без времени
- `LocalTime` — время без даты
- `LocalDateTime` — дата и время без часового пояса
- `ZonedDateTime` — дата и время с часовым поясом
- `Instant` — момент времени в UTC
- `Duration` — длительность в секундах/наносекундах
- `Period` — период в датах (годы, месяцы, дни)

---

## LocalDate

Представляет дату: год, месяц, день. Без времени и часового пояса.

```java
LocalDate today = LocalDate.now();                    // текущая дата
LocalDate date = LocalDate.of(2025, 6, 15);           // конкретная дата
LocalDate parsed = LocalDate.parse("2025-06-15");     // из строки ISO-8601

// Получение полей:
date.getYear();        // 2025
date.getMonth();       // Month.JUNE
date.getMonthValue();  // 6
date.getDayOfMonth();  // 15
date.getDayOfWeek();   // DayOfWeek.SUNDAY

// Арифметика:
date.plusDays(10);
date.minusMonths(2);
date.plusYears(1);
date.withDayOfMonth(1); // первый день того же месяца

// Сравнение:
date.isBefore(LocalDate.of(2026, 1, 1));  // true
date.isAfter(LocalDate.of(2020, 1, 1));   // true
date.isEqual(today);
```

---

## LocalTime

Представляет время суток: часы, минуты, секунды, наносекунды.

```java
LocalTime now = LocalTime.now();
LocalTime time = LocalTime.of(14, 30, 0);         // 14:30:00
LocalTime parsed = LocalTime.parse("14:30:00");

time.getHour();    // 14
time.getMinute();  // 30
time.getSecond();  // 0

time.plusHours(2);       // 16:30:00
time.minusMinutes(15);   // 14:15:00
time.withHour(9);        // 09:30:00

// Граничные значения:
LocalTime.MIDNIGHT;  // 00:00:00
LocalTime.NOON;      // 12:00:00
```

---

## LocalDateTime

Объединяет дату и время, но без информации о часовом поясе.

```java
LocalDateTime now = LocalDateTime.now();
LocalDateTime dt = LocalDateTime.of(2025, 6, 15, 14, 30);
LocalDateTime dt2 = LocalDateTime.of(LocalDate.of(2025, 6, 15), LocalTime.of(14, 30));

// Разобрать/собрать:
dt.toLocalDate();   // LocalDate
dt.toLocalTime();   // LocalTime

dt.plusDays(5).minusHours(2);
```

---

## ZonedDateTime

Дата и время с привязкой к часовому поясу. Учитывает переход на летнее/зимнее время.

```java
ZonedDateTime zdt = ZonedDateTime.now();
ZonedDateTime moscow = ZonedDateTime.now(ZoneId.of("Europe/Moscow"));
ZonedDateTime utc = ZonedDateTime.now(ZoneOffset.UTC);

// Конвертация между зонами:
ZonedDateTime inTokyo = moscow.withZoneSameInstant(ZoneId.of("Asia/Tokyo"));

// Получение информации:
zdt.getZone();    // ZoneId
zdt.getOffset();  // ZoneOffset, например +03:00

// Список доступных зон:
ZoneId.getAvailableZoneIds(); // Set<String>
```

---

## Instant

Момент времени — количество секунд (и наносекунд) с `1970-01-01T00:00:00Z` (Unix epoch). Не привязан к часовому поясу — просто точка на временной шкале.

```java
Instant now = Instant.now();
Instant epoch = Instant.EPOCH;                         // 1970-01-01T00:00:00Z
Instant fromEpoch = Instant.ofEpochSecond(1_000_000);

now.toEpochMilli();   // миллисекунды с эпохи
now.getEpochSecond(); // секунды с эпохи

// Конвертация:
Instant instant = ZonedDateTime.now().toInstant();
ZonedDateTime back = instant.atZone(ZoneId.of("Europe/Moscow"));

// Сравнение:
instant.isBefore(Instant.now());
instant.isAfter(Instant.EPOCH);
```

> `Instant` удобен для хранения временных меток (timestamp) в базах данных и логах.

---

## Duration

Измеряет промежуток времени в секундах и наносекундах. Используется с `LocalTime`, `Instant`, `ZonedDateTime`.

```java
Duration d1 = Duration.ofHours(2);
Duration d2 = Duration.ofMinutes(90);
Duration d3 = Duration.ofSeconds(3600);
Duration d4 = Duration.parse("PT1H30M"); // ISO-8601: 1 час 30 минут

// Между двумя моментами:
LocalTime start = LocalTime.of(10, 0);
LocalTime end = LocalTime.of(12, 30);
Duration between = Duration.between(start, end); // PT2H30M

between.toHours();    // 2
between.toMinutes();  // 150
between.toSeconds();  // 9000

// Арифметика:
d1.plus(d2);
d1.minus(Duration.ofMinutes(30));
d1.multipliedBy(3);
d1.isNegative();
d1.isZero();
```

---

## Period

Измеряет календарный промежуток в годах, месяцах и днях. Используется с `LocalDate`.

```java
Period p1 = Period.ofYears(1);
Period p2 = Period.ofMonths(6);
Period p3 = Period.of(1, 2, 3); // 1 год, 2 месяца, 3 дня
Period p4 = Period.parse("P1Y2M3D");

// Между двумя датами:
LocalDate birth = LocalDate.of(2000, 6, 15);
LocalDate today = LocalDate.now();
Period age = Period.between(birth, today);

age.getYears();   // полных лет
age.getMonths();  // месяцев сверх лет
age.getDays();    // дней сверх месяцев

// Применение к дате:
LocalDate future = today.plus(Period.ofMonths(6));
```

### Duration vs Period

| `Duration`    | `Period`                                |                              |
| ------------- | --------------------------------------- | ---------------------------- |
| Единицы       | секунды, наносекунды                    | годы, месяцы, дни            |
| Применяется к | `Instant`, `LocalTime`, `ZonedDateTime` | `LocalDate`, `LocalDateTime` |
| Учитывает DST | нет                                     | да (через `ZonedDateTime`)   |
| Парсинг       | `PT1H30M`                               | `P1Y2M3D`                    |

## DateTimeFormatter

Для форматирования и парсинга дат используется `DateTimeFormatter`.

```java
// Встроенные форматтеры:
DateTimeFormatter iso = DateTimeFormatter.ISO_LOCAL_DATE;         // "2025-06-15"
DateTimeFormatter isoTime = DateTimeFormatter.ISO_LOCAL_DATE_TIME; // "2025-06-15T14:30:00"

// Собственный паттерн:
DateTimeFormatter fmt = DateTimeFormatter.ofPattern("dd.MM.yyyy HH:mm");

LocalDateTime dt = LocalDateTime.of(2025, 6, 15, 14, 30);
String formatted = dt.format(fmt);              // "15.06.2025 14:30"
LocalDateTime parsed = LocalDateTime.parse("15.06.2025 14:30", fmt);

// С локалью:
DateTimeFormatter ruFmt = DateTimeFormatter
    .ofPattern("d MMMM yyyy", new Locale("ru"));
LocalDate.of(2025, 6, 15).format(ruFmt); // "15 июня 2025"
```

### Символы паттернов

|Символ|Значение|Пример|
|---|---|---|
|`yyyy`|год (4 цифры)|2025|
|`MM`|месяц (2 цифры)|06|
|`dd`|день (2 цифры)|15|
|`HH`|часы 24h|14|
|`mm`|минуты|30|
|`ss`|секунды|00|
|`SSS`|миллисекунды|123|
|`z`|зона (текст)|MSK|
|`Z`|зона (смещение)|+0300|

---

## Важные особенности

- Классы **immutable** — любая операция возвращает новый объект, исходный не меняется.
- Хорошо поддерживают операции `plus`, `minus`, `with`.
- Для форматирования используется `DateTimeFormatter`.
- `ZonedDateTime` корректно обрабатывает переход на летнее/зимнее время (DST).
- `Instant` — лучший выбор для хранения временных меток.

---

## Шпаргалка

- `java.time` — современный пакет даты и времени.
- `LocalDate` — только дата; `LocalTime` — только время; `LocalDateTime` — оба без зоны.
- `ZonedDateTime` — дата + время + часовой пояс; корректен при DST.
- `Instant` — точка на временной шкале UTC; удобен для timestamp.
- `Duration` — в секундах/наносекундах; `Period` — в годах/месяцах/днях.
- Все классы immutable — арифметика возвращает новый объект.
- `DateTimeFormatter` — для форматирования и парсинга.
- `Duration.between` — для времени; `Period.between` — для дат.
