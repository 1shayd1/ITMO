# 43. Форматирование локализованных числовых данных, текста, даты и времени

**Форматирование с учётом локали** — преобразование внутреннего представления данных (числа, даты, валюты) в строку, понятную пользователю из конкретной культурной среды, и обратный парсинг строки в данные.

---

## 🌍 Locale — основа форматирования

`Locale` определяет правила форматирования: разделители тысяч и дробной части, формат даты, символ валюты, названия месяцев и дней недели.

```java
Locale us = Locale.US;           // США: 1,234.56, 12/31/2025, $1,234.56
Locale germany = Locale.GERMANY; // Германия: 1.234,56, 31.12.2025, 1.234,56 €
Locale russia = new Locale("ru", "RU"); // Россия: 1 234,56, 31.12.2025, 1 234,56 ₽
```

---

## 🔢 1. Форматирование чисел (NumberFormat)

### Основные типы форматирования

| Метод | Назначение | Пример (Locale.US) | Пример (Locale.GERMANY) |
|-------|------------|-------------------|------------------------|
| `getInstance()` | Общий числовой формат | `1,234.56` | `1.234,56` |
| `getNumberInstance()` | То же, что `getInstance()` | `1,234.56` | `1.234,56` |
| `getIntegerInstance()` | Только целая часть | `1,235` (округление) | `1.235` |
| `getCurrencyInstance()` | Валюта | `$1,234.56` | `1.234,56 €` |
| `getPercentInstance()` | Проценты | `123,456%` | `123.456%` |
| `getCompactNumberInstance()` (Java 12+) | Компактная форма | `1.2K` | `1,2 Tsd.` |

### Примеры форматирования

```java
double number = 1234567.89;

// Общий числовой формат
NumberFormat nfUS = NumberFormat.getInstance(Locale.US);
NumberFormat nfDE = NumberFormat.getInstance(Locale.GERMANY);
NumberFormat nfRU = NumberFormat.getInstance(new Locale("ru", "RU"));

System.out.println(nfUS.format(number)); // 1,234,567.89
System.out.println(nfDE.format(number)); // 1.234.567,89
System.out.println(nfRU.format(number)); // 1 234 567,89

// Валюта
NumberFormat cfUS = NumberFormat.getCurrencyInstance(Locale.US);
NumberFormat cfDE = NumberFormat.getCurrencyInstance(Locale.GERMANY);
NumberFormat cfRU = NumberFormat.getCurrencyInstance(new Locale("ru", "RU"));
NumberFormat cfJP = NumberFormat.getCurrencyInstance(Locale.JAPAN);

System.out.println(cfUS.format(number)); // $1,234,567.89
System.out.println(cfDE.format(number)); // 1.234.567,89 €
System.out.println(cfRU.format(number)); // 1 234 567,89 ₽
System.out.println(cfJP.format(number)); // ￥1,234,568 (округление)

// Проценты (0.1 = 10%)
NumberFormat pfUS = NumberFormat.getPercentInstance(Locale.US);
NumberFormat pfDE = NumberFormat.getPercentInstance(Locale.GERMANY);

System.out.println(pfUS.format(0.1234)); // 12%
System.out.println(pfDE.format(0.1234)); // 12 %
```

### Парсинг чисел из локализованных строк

```java
// Обратное преобразование: строка → число
NumberFormat nfDE = NumberFormat.getInstance(Locale.GERMANY);
try {
    Number parsed = nfDE.parse("1.234.567,89");
    System.out.println(parsed.doubleValue()); // 1234567.89
} catch (ParseException e) {
    e.printStackTrace();
}

// Парсинг валюты
NumberFormat cfDE = NumberFormat.getCurrencyInstance(Locale.GERMANY);
Number amount = cfDE.parse("1.234,56 €");
System.out.println(amount.doubleValue()); // 1234.56
```

### Настройка форматирования

```java
NumberFormat nf = NumberFormat.getInstance(Locale.US);
nf.setMaximumFractionDigits(2);
nf.setMinimumFractionDigits(2);
nf.setGroupingUsed(true);      // разделители тысяч
System.out.println(nf.format(1234.5)); // 1,234.50

// Компактный формат (Java 12+)
NumberFormat compactUS = NumberFormat.getCompactNumberInstance(Locale.US, NumberFormat.Style.SHORT);
NumberFormat compactRU = NumberFormat.getCompactNumberInstance(new Locale("ru"), NumberFormat.Style.SHORT);

System.out.println(compactUS.format(1500));   // 1.5K
System.out.println(compactUS.format(2500000)); // 2.5M
System.out.println(compactRU.format(1500));   // 1,5 тыс.
System.out.println(compactRU.format(2500000)); // 2,5 млн
```

## 💰 2. Форматирование валют (особенности)

```java
// Разные валюты для одной локали
NumberFormat usdFormat = NumberFormat.getCurrencyInstance(Locale.US);
NumberFormat eurFormat = NumberFormat.getCurrencyInstance(Locale.GERMANY);
NumberFormat jpyFormat = NumberFormat.getCurrencyInstance(Locale.JAPAN);

double dollars = 1234.56;
System.out.println(usdFormat.format(dollars)); // $1,234.56
System.out.println(eurFormat.format(dollars)); // 1.234,56 €
System.out.println(jpyFormat.format(dollars)); // ￥1,235 (иена без копеек)

// Получение информации о валюте
Currency currency = Currency.getInstance(Locale.US);
System.out.println(currency.getCurrencyCode()); // USD
System.out.println(currency.getSymbol());       // $
System.out.println(currency.getDisplayName());  // US Dollar
```

### DecimalFormat — полный контроль над шаблоном

```java
// Кастомный формат для конкретной задачи
DecimalFormat df = new DecimalFormat("#,##0.00");
System.out.println(df.format(1234.567)); // 1,234.57

// Символ валюты в шаблоне
DecimalFormat cf = new DecimalFormat("¤ #,##0.00");
cf.setCurrency(Currency.getInstance(Locale.US));
System.out.println(cf.format(1234.56)); // $ 1,234.56
```

### Шаблоны DecimalFormat

| Символ | Значение | Пример |
|--------|----------|--------|
| `0` | Цифра (обязательная, ведущие нули) | `000.00` → `012.34` |
| `#` | Цифра (опциональная) | `#.##` → `12.34` |
| `.` | Разделитель целой и дробной части | `#.#` |
| `,` | Разделитель групп (тысяч) | `#,###` |
| `¤` | Символ валюты | `¤#.00` → `$12.34` |
| `%` | Проценты (умножение на 100) | `#%` → `12%` |
| `;` | Разделитель положительных/отрицательных шаблонов | `#;#-` |

## 📅 3. Форматирование даты и времени

### Старый API (DateFormat) — для поддержки legacy кода

```java
Date now = new Date();

// Только дата
DateFormat dfUS = DateFormat.getDateInstance(DateFormat.LONG, Locale.US);
DateFormat dfDE = DateFormat.getDateInstance(DateFormat.LONG, Locale.GERMANY);
DateFormat dfRU = DateFormat.getDateInstance(DateFormat.LONG, new Locale("ru", "RU"));

System.out.println(dfUS.format(now)); // January 1, 2025
System.out.println(dfDE.format(now)); // 1. Januar 2025
System.out.println(dfRU.format(now)); // 1 января 2025 г.

// Только время
DateFormat tfUS = DateFormat.getTimeInstance(DateFormat.MEDIUM, Locale.US);
DateFormat tfDE = DateFormat.getTimeInstance(DateFormat.MEDIUM, Locale.GERMANY);
System.out.println(tfUS.format(now)); // 3:45:12 PM
System.out.println(tfDE.format(now)); // 15:45:12

// Дата + время
DateFormat dtfUS = DateFormat.getDateTimeInstance(DateFormat.FULL, DateFormat.FULL, Locale.US);
System.out.println(dtfUS.format(now)); 
// Thursday, January 1, 2025 at 3:45:12 PM Eastern Standard Time
```

### Стили DateFormat

| Стиль | en-US (дата) | ru-RU (дата) |
|-------|--------------|--------------|
| `SHORT` | 1/1/25 | 01.01.25 |
| `MEDIUM` | Jan 1, 2025 | 1 янв. 2025 г. |
| `LONG` | January 1, 2025 | 1 января 2025 г. |
| `FULL` | Thursday, January 1, 2025 | четверг, 1 января 2025 г. |

### Парсинг дат

```java
DateFormat dfRU = DateFormat.getDateInstance(DateFormat.SHORT, new Locale("ru", "RU"));
try {
    Date date = dfRU.parse("31.12.2025");
    System.out.println(date);
} catch (ParseException e) {
    e.printStackTrace();
}
```

## ⏰ 4. Форматирование даты и времени (Java 8+ DateTimeFormatter)

Современный API — **thread-safe**, работает с `LocalDate`, `LocalTime`, `LocalDateTime`, `ZonedDateTime`.

```java
LocalDate date = LocalDate.of(2025, 12, 31);
LocalTime time = LocalTime.of(14, 30, 15);
LocalDateTime dateTime = LocalDateTime.of(date, time);
ZonedDateTime zoned = ZonedDateTime.now(ZoneId.of("Europe/Moscow"));

// Предопределённые локализованные форматы
DateTimeFormatter dfUS = DateTimeFormatter.ofLocalizedDate(FormatStyle.LONG).withLocale(Locale.US);
DateTimeFormatter dfDE = DateTimeFormatter.ofLocalizedDate(FormatStyle.LONG).withLocale(Locale.GERMANY);
DateTimeFormatter dfRU = DateTimeFormatter.ofLocalizedDate(FormatStyle.LONG).withLocale(new Locale("ru", "RU"));

System.out.println(dfUS.format(date));  // December 31, 2025
System.out.println(dfDE.format(date));  // 31. Dezember 2025
System.out.println(dfRU.format(date));  // 31 декабря 2025 г.

// Время
DateTimeFormatter tfUS = DateTimeFormatter.ofLocalizedTime(FormatStyle.MEDIUM).withLocale(Locale.US);
DateTimeFormatter tfDE = DateTimeFormatter.ofLocalizedTime(FormatStyle.MEDIUM).withLocale(Locale.GERMANY);
System.out.println(tfUS.format(time));  // 2:30:15 PM
System.out.println(tfDE.format(time));  // 14:30:15

// Дата + время
DateTimeFormatter dtfUS = DateTimeFormatter.ofLocalizedDateTime(FormatStyle.FULL, FormatStyle.FULL)
    .withLocale(Locale.US);
System.out.println(dtfUS.format(zoned)); 
// Thursday, December 31, 2025 at 2:30:15 PM Moscow Standard Time

// Парсинг
LocalDate parsedDate = LocalDate.parse("31 декабря 2025 г.", dfRU);
System.out.println(parsedDate); // 2025-12-31
```

### Кастомные паттерны (языконезависимые)

```java
// Паттерны одинаковы для всех локалей (но названия месяцев будут на языке локали)
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("dd MMMM yyyy, EEEE", Locale.US);
System.out.println(formatter.format(date)); // 31 December 2025, Wednesday

formatter = DateTimeFormatter.ofPattern("dd MMMM yyyy, EEEE", new Locale("ru"));
System.out.println(formatter.format(date)); // 31 декабря 2025, среда

// ISO-форматы
System.out.println(date.format(DateTimeFormatter.ISO_DATE));         // 2025-12-31
System.out.println(dateTime.format(DateTimeFormatter.ISO_DATE_TIME)); // 2025-12-31T14:30:15
```

### Таблица паттернов DateTimeFormatter

| Символ | Значение | Пример (ru-RU) |
|--------|----------|----------------|
| `yyyy` | Год (4 цифры) | 2025 |
| `yy` | Год (2 цифры) | 25 |
| `MM` | Месяц (цифра, 2 позиции) | 12 |
| `MMM` | Месяц (аббр.) | дек. |
| `MMMM` | Месяц (полный) | декабря |
| `dd` | День месяца | 31 |
| `EE` | День недели (аббр.) | ср |
| `EEEE` | День недели (полный) | среда |
| `HH` | Час (0-23) | 14 |
| `hh` | Час (1-12) | 02 |
| `mm` | Минуты | 30 |
| `ss` | Секунды | 15 |
| `a` | AM/PM | PM (для en-US) |
| `z` | Часовой пояс (аббр.) | MSK |
| `zzzz` | Часовой пояс (полный) | Moscow Standard Time |

## 📝 5. Форматирование текста (MessageFormat)

`MessageFormat` позволяет подставлять значения в шаблоны с автоматическим форматированием под локаль.

```java
// Базовое использование
String template = "User {0} logged in at {1,time,short} on {2,date,long}";
String result = MessageFormat.format(template, "john_doe", new Date(), new Date());
System.out.println(result); 
// User john_doe logged in at 3:45 PM on January 1, 2025

// С разными локалями
MessageFormat mfUS = new MessageFormat("Price: {0,number,currency}", Locale.US);
MessageFormat mfDE = new MessageFormat("Price: {0,number,currency}", Locale.GERMANY);

System.out.println(mfUS.format(new Object[]{1234.56})); // Price: $1,234.56
System.out.println(mfDE.format(new Object[]{1234.56})); // Price: 1.234,56 €

// Сложный пример с ChoiceFormat
String pluralPattern = "{0,choice,0#no messages|1#one message|1<{0} messages}";
String msg = MessageFormat.format(pluralPattern, 5);
System.out.println(msg); // 5 messages
```

### Синтаксис форматов в MessageFormat

```
{аргумент, тип, стиль}
```

| Тип | Стили | Пример |
|-----|-------|--------|
| `number` | `integer`, `currency`, `percent` | `{0,number,currency}` |
| `date` | `short`, `medium`, `long`, `full` | `{1,date,long}` |
| `time` | `short`, `medium`, `long`, `full` | `{2,time,short}` |
| `choice` | Диапазоны | `{0,choice,0#none|1#one|1<many}` |

## 🧠 Полный пример: форматирование пользовательских данных

```java
import java.text.*;
import java.time.*;
import java.time.format.*;
import java.util.*;

public class UserProfileFormatter {
    
    public static void displayUserInfo(Locale locale) {
        ResourceBundle bundle = ResourceBundle.getBundle("profile", locale);
        
        // Данные пользователя
        String name = "John Doe";
        double balance = 12345.67;
        LocalDate birthDate = LocalDate.of(1990, 5, 15);
        LocalTime loginTime = LocalTime.of(14, 30, 0);
        int messageCount = 3;
        
        // 1. Простое приветствие
        System.out.println(bundle.getString("welcome") + ", " + name + "!");
        
        // 2. Баланс (валюта)
        NumberFormat currencyFormat = NumberFormat.getCurrencyInstance(locale);
        System.out.println(bundle.getString("balance") + ": " + currencyFormat.format(balance));
        
        // 3. Дата рождения
        DateTimeFormatter dateFormat = DateTimeFormatter.ofLocalizedDate(FormatStyle.LONG)
            .withLocale(locale);
        System.out.println(bundle.getString("birthdate") + ": " + birthDate.format(dateFormat));
        
        // 4. Время последнего входа
        DateTimeFormatter timeFormat = DateTimeFormatter.ofLocalizedTime(FormatStyle.MEDIUM)
            .withLocale(locale);
        System.out.println(bundle.getString("lastLogin") + ": " + loginTime.format(timeFormat));
        
        // 5. Количество сообщений (ChoiceFormat)
        String pattern = bundle.getString("messages");
        String messages = MessageFormat.format(pattern, messageCount);
        System.out.println(messages);
        
        System.out.println("---");
    }
    
    public static void main(String[] args) {
        displayUserInfo(Locale.US);
        displayUserInfo(Locale.GERMANY);
        displayUserInfo(new Locale("ru", "RU"));
    }
}
```

**profile_en.properties**
```properties
welcome=Welcome
balance=Balance
birthdate=Birth date
lastLogin=Last login
messages=You have {0,choice,0#no messages|1#one message|1<{0} messages}
```

**profile_de.properties**
```properties

balance=Kontostand
birthdate=Geburtsdatum
lastLogin=Letzte Anmeldung
messages=Sie haben {0,choice,0#keine Nachrichten|1#eine Nachricht|1<{0} Nachrichten}
```

**profile_ru.properties**
```properties
welcome=Добро пожаловать
balance=Баланс
birthdate=Дата рождения
lastLogin=Последний вход
messages=У вас {0,choice,0#нет сообщений|1#одно сообщение|1<{0} сообщений}
```

## ⚠️ Частые ошибки

### ❌ Неправильно

```java
// 1. Конкатенация чисел со строками
String price = "Price: " + 1234.56;  // Потеря локализации

// 2. Ручное форматирование даты
String date = day + "." + month + "." + year;  // Неправильно для других стран

// 3. Парсинг чисел без локали
double val = Double.parseDouble("1.234,56");  // NumberFormatException

// 4. Переиспользование не-thread-safe форматтеров
private static NumberFormat sharedFormat = NumberFormat.getInstance(Locale.US);
// Использование из нескольких потоков -> проблемы
```

### ✅ Правильно

```java
// 1. Использовать NumberFormat
String price = NumberFormat.getCurrencyInstance(locale).format(1234.56);

// 2. Использовать DateTimeFormatter
String date = LocalDate.now().format(DateTimeFormatter.ofLocalizedDate(FormatStyle.MEDIUM));

// 3. Парсить через NumberFormat
NumberFormat nf = NumberFormat.getInstance(Locale.GERMANY);
double val = nf.parse("1.234,56").doubleValue();

// 4. Создавать форматтеры при необходимости или использовать ThreadLocal
private static final ThreadLocal<NumberFormat> formatter = 
    ThreadLocal.withInitial(() -> NumberFormat.getInstance(Locale.US));
```

## 📊 Сравнение подходов

| Характеристика | DateFormat (старый) | DateTimeFormatter (Java 8+) |
|----------------|---------------------|----------------------------|
| Thread-safety | ❌ | ✅ |
| Работа с `Date` | ✅ | ⚠️ (нужен конвертер) |
| Работа с `LocalDate` | ❌ | ✅ |
| Поддержка паттернов | ✅ | ✅ |
| Локализация | ✅ | ✅ |
| Производительность | Средняя | Высокая |
| Рекомендация | Только legacy | ✅ **Первый выбор** |

## 🔗 Связи с другими темами

- `Locale` — основа для всех форматтеров
- `ResourceBundle` — хранение локализованных шаблонов
- `MessageFormat` — комбинирует текст + числа + даты
- `ChoiceFormat` — множественные формы
- `DecimalFormat` — кастомные шаблоны чисел
- `SimpleDateFormat` — кастомные шаблоны дат (но лучше `DateTimeFormatter`)

---

### Шпаргалка

* **Числа** → `NumberFormat.getInstance(locale)`
* **Валюты** → `NumberFormat.getCurrencyInstance(locale)`
* **Проценты** → `NumberFormat.getPercentInstance(locale)`
* **Даты/время (Java 8+)** → `DateTimeFormatter.ofLocalizedXxx().withLocale(locale)`
* **Текст + подстановки** → `MessageFormat`
* **Locale** — везде: `Locale.US`, `Locale.GERMANY`, `new Locale("ru", "RU")`
* **DateTimeFormatter** thread-safe → можно хранить как статическую константу
* **DateFormat** НЕ thread-safe → создавайте на каждый вызов
* **Парсинг** всегда используйте тот же форматтер, которым форматировали
* **Никогда не собирайте локализованные строки вручную** — используйте готовые классы
