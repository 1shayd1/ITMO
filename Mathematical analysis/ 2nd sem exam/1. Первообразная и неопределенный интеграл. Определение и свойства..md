# Первообразная и неопределённый интеграл. Свойства. Таблица интегралов. Интегрирование по частям. Замена переменной в определённом интеграле

## Первообразная

**Определение.** Функция $F(x)$ называется **первообразной** для функции $f(x)$ на промежутке $X$, если $F(x)$ дифференцируема на $X$ и для всех $x \in X$ выполняется равенство $$F'(x) = f(x).$$

**Теорема (об общем виде первообразных).** Если $F(x)$ — первообразная для $f(x)$ на промежутке $X$, то:

1. для любой постоянной $C$ функция $F(x)+C$ также является первообразной для $f(x)$;
2. всякая первообразная $\Phi(x)$ для $f(x)$ на $X$ представима в виде $\Phi(x) = F(x) + C$ для некоторой постоянной $C$.

**Доказательство.**

_1._ Пусть $G(x) = F(x) + C$. Тогда $G'(x) = F'(x) + 0 = f(x)$, значит $G(x)$ — первообразная для $f(x)$.

_2._ Пусть $\Phi(x)$ — другая первообразная для $f(x)$ на $X$, то есть $\Phi'(x) = f(x)$ для всех $x \in X$. Рассмотрим функцию $$\psi(x) = \Phi(x) - F(x).$$ Тогда $$\psi'(x) = \Phi'(x) - F'(x) = f(x) - f(x) = 0$$ для всех $x \in X$.

Покажем, что функция, производная которой тождественно равна нулю на промежутке, является постоянной. Возьмём произвольные точки $x_1, x_2 \in X$, $x_1 < x_2$. По теореме Лагранжа о среднем значении найдётся точка $c \in (x_1, x_2)$ такая, что $$\psi(x_2) - \psi(x_1) = \psi'(c)(x_2 - x_1) = 0 \cdot (x_2-x_1) = 0,$$ следовательно $\psi(x_2) = \psi(x_1)$. В силу произвольности $x_1, x_2$ функция $\psi(x)$ постоянна на $X$, то есть $\psi(x) \equiv C$ для некоторой константы $C$. Отсюда $$\Phi(x) = F(x) + C. \qquad \blacksquare$$

**Замечание о существенности условия "промежуток".** Если $X$ не является промежутком (например, состоит из нескольких непересекающихся интервалов), утверждение неверно — на каждом интервале константа может быть своей.

## Неопределённый интеграл

**Определение.** Совокупность всех первообразных функции $f(x)$ на промежутке $X$ называется **неопределённым интегралом** от функции $f(x)$ и обозначается $$\int f(x),dx = F(x) + C,$$ где $F(x)$ — какая-либо первообразная для $f(x)$, а $C$ — произвольная постоянная.

Функция $f(x)$ называется **подынтегральной функцией**, выражение $f(x),dx$ — **подынтегральным выражением**.

## Свойства неопределённого интеграла

**1.** $\left(\int f(x),dx\right)' = f(x)$.

**2.** $d\left(\int f(x),dx\right) = f(x),dx$.

**3.** $\int dF(x) = F(x) + C$, то есть $\int F'(x),dx = F(x) + C$.

**4.** Линейность: если существуют $\int f(x),dx$ и $\int g(x),dx$, то для любых констант $\alpha, \beta$ (не равных одновременно нулю) существует $$\int \big(\alpha f(x) + \beta g(x)\big),dx = \alpha \int f(x),dx + \beta \int g(x),dx.$$

**Доказательство свойства 4.** Пусть $F'(x)=f(x)$, $G'(x)=g(x)$. Рассмотрим функцию $\alpha F(x) + \beta G(x)$. По правилам дифференцирования $$\big(\alpha F(x) + \beta G(x)\big)' = \alpha F'(x) + \beta G'(x) = \alpha f(x) + \beta g(x).$$ Следовательно, $\alpha F(x) + \beta G(x)$ является первообразной для $\alpha f(x) + \beta g(x)$, а значит $$\int(\alpha f(x)+\beta g(x)),dx = \alpha F(x)+\beta G(x) + C = \alpha\int f(x),dx + \beta\int g(x),dx$$ (с точностью до объединения произвольных констант в одну). $\blacksquare$

**Доказательство свойств 1–3.** Свойство 1 — прямое следствие определения первообразной: $F'(x)=f(x)$, и операция $\int \cdot ,dx$ возвращает $F(x)+C$, производная которого равна $f(x)$.

Свойство 2 следует из 1: $d\left(\int f,dx\right) = \left(\int f,dx\right)'dx = f(x),dx$.

Свойство 3: по определению $F(x)$ — первообразная для $F'(x)$ (поскольку $(F(x))'=F'(x)$), значит $\int F'(x),dx = F(x)+C$. $\blacksquare$

## Таблица основных интегралов

$$\int x^\alpha,dx = \frac{x^{\alpha+1}}{\alpha+1}+C \quad (\alpha \ne -1)$$ $$\int \frac{dx}{x} = \ln|x|+C$$ $$\int a^x,dx = \frac{a^x}{\ln a}+C \quad (a>0,\ a\ne1), \qquad \int e^x,dx=e^x+C$$ $$\int \sin x,dx = -\cos x + C, \qquad \int \cos x,dx = \sin x + C$$ $$\int \frac{dx}{\cos^2 x} = \tan x + C, \qquad \int \frac{dx}{\sin^2 x} = -\cot x + C$$ $$\int \frac{dx}{\sqrt{a^2-x^2}} = \arcsin\frac{x}{a}+C \quad (a>0)$$ $$\int \frac{dx}{a^2+x^2} = \frac{1}{a}\arctan\frac{x}{a}+C \quad (a\ne0)$$ $$\int \frac{dx}{\sqrt{x^2\pm a^2}} = \ln\left|x+\sqrt{x^2\pm a^2}\right|+C$$ $$\int \frac{dx}{a^2-x^2} = \frac{1}{2a}\ln\left|\frac{a+x}{a-x}\right|+C$$ $$\int \sinh x,dx = \cosh x + C, \qquad \int \cosh x,dx = \sinh x + C$$

**Доказательство одной из формул (на примере $\int \frac{dx}{a^2+x^2}$).** Проверим прямым дифференцированием: $$\left(\frac{1}{a}\arctan\frac{x}{a}\right)' = \frac{1}{a}\cdot\frac{1}{1+(x/a)^2}\cdot\frac{1}{a} = \frac{1}{a^2}\cdot\frac{a^2}{a^2+x^2} = \frac{1}{a^2+x^2}.$$ Значит, $\frac{1}{a}\arctan\frac{x}{a}$ — первообразная для $\frac{1}{a^2+x^2}$, что и требовалось. $\blacksquare$

Аналогичной проверкой дифференцированием устанавливаются все остальные формулы таблицы — каждая из них следует из соответствующей формулы дифференцирования элементарных функций.

## Интегрирование по частям

**Теорема.** Пусть функции $u(x)$ и $v(x)$ дифференцируемы на промежутке $X$, и существует интеграл $\int v(x)u'(x),dx$. Тогда существует и интеграл $\int u(x)v'(x),dx$, причём $$\int u(x)v'(x),dx = u(x)v(x) - \int v(x)u'(x),dx,$$ или в краткой форме $$\int u,dv = uv - \int v,du.$$

**Доказательство.** По правилу дифференцирования произведения $$(u(x)v(x))' = u'(x)v(x) + u(x)v'(x).$$ Проинтегрируем обе части равенства (используя свойство 3 и линейность интеграла): $$u(x)v(x) + C_1 = \int u'(x)v(x),dx + \int u(x)v'(x),dx.$$ По условию правый первый интеграл существует, значит существует и второй, и $$\int u(x)v'(x),dx = u(x)v(x) - \int u'(x)v(x),dx,$$ где произвольная постоянная учитывается в правом интеграле. $\blacksquare$

**Геометрический и практический смысл.** Метод сводит интегрирование произведения $u,dv$ к интегрированию "обратного" произведения $v,du$, что часто проще, особенно когда одна из функций — многочлен (его степень понижается при дифференцировании), а другая легко интегрируется (например $e^x, \sin x, \cos x$).

## Замена переменной (подстановка) в определённом интеграле

**Теорема.** Пусть функция $f(x)$ непрерывна на отрезке $[a,b]$, а функция $x=\varphi(t)$ непрерывно дифференцируема на отрезке $[\alpha,\beta]$, причём $\varphi(\alpha)=a$, $\varphi(\beta)=b$ и $\varphi(t)\in[a,b]$ для всех $t\in[\alpha,\beta]$. Тогда $$\int_a^b f(x),dx = \int_\alpha^\beta f(\varphi(t))\varphi'(t),dt.$$

**Доказательство.** Так как $f$ непрерывна на $[a,b]$, она имеет первообразную. Пусть $F(x)$ — первообразная для $f(x)$ на $[a,b]$, то есть $F'(x)=f(x)$. По формуле Ньютона–Лейбница (которая будет строго доказана в билете 9, но используется здесь для построения доказательства) $$\int_a^b f(x),dx = F(b)-F(a).$$

Рассмотрим сложную функцию $\Phi(t) = F(\varphi(t))$. По правилу дифференцирования сложной функции, поскольку $F$ дифференцируема в точках $\varphi(t)$ и $\varphi$ дифференцируема по $t$: $$\Phi'(t) = F'(\varphi(t))\cdot \varphi'(t) = f(\varphi(t))\varphi'(t).$$

Функция $\Phi'(t)=f(\varphi(t))\varphi'(t)$ непрерывна на $[\alpha,\beta]$ (как произведение непрерывных функций — $f\circ\varphi$ непрерывна как композиция непрерывных функций, $\varphi'$ непрерывна по условию). Следовательно, $\Phi(t)$ — первообразная для $f(\varphi(t))\varphi'(t)$, и по формуле Ньютона–Лейбница $$\int_\alpha^\beta f(\varphi(t))\varphi'(t),dt = \Phi(\beta)-\Phi(\alpha) = F(\varphi(\beta)) - F(\varphi(\alpha)) = F(b)-F(a).$$

Сравнивая два выражения, получаем $$\int_a^b f(x),dx = F(b)-F(a) = \int_\alpha^\beta f(\varphi(t))\varphi'(t),dt. \qquad \blacksquare$$

**Практическое следствие.** При вычислении определённого интеграла заменой переменной **не нужно возвращаться к старой переменной** — достаточно пересчитать пределы интегрирования по формулам $\varphi(\alpha)=a$, $\varphi(\beta)=b$.

## Примеры

**Пример 1 (применение таблицы интегралов и линейности).**

Вычислить $\displaystyle\int (3x^2 - \frac{2}{x} + \cos x),dx$.

По свойству линейности: $$\int 3x^2,dx - \int \frac{2}{x},dx + \int \cos x,dx = 3\cdot\frac{x^3}{3} - 2\ln|x| + \sin x + C = x^3 - 2\ln|x| + \sin x + C.$$

**Пример 2 (интегрирование по частям, $u$ — многочлен).**

Вычислить $\displaystyle\int x e^x,dx$.

Пусть $u=x$, $dv=e^x,dx$. Тогда $du=dx$, $v=e^x$. По формуле: $$\int x e^x,dx = xe^x - \int e^x,dx = xe^x - e^x + C = e^x(x-1)+C.$$

**Пример 3 (интегрирование по частям, "циклический" случай).**

Вычислить $\displaystyle\int e^x \cos x,dx$.

Пусть $u=e^x$, $dv=\cos x,dx$, тогда $du=e^xdx$, $v=\sin x$: $$I=\int e^x\cos x,dx = e^x\sin x - \int e^x\sin x,dx.$$ Применим интегрирование по частям ещё раз к $\int e^x\sin x,dx$: пусть $u=e^x$, $dv=\sin x,dx$, $du=e^xdx$, $v=-\cos x$: $$\int e^x\sin x,dx = -e^x\cos x + \int e^x\cos x,dx = -e^x\cos x + I.$$ Подставим: $$I = e^x\sin x - (-e^x\cos x + I) = e^x\sin x + e^x\cos x - I.$$ $$2I = e^x(\sin x+\cos x) \implies I = \frac{e^x(\sin x+\cos x)}{2}+C.$$

**Пример 4 (замена переменной в определённом интеграле).**

Вычислить $\displaystyle\int_0^{\pi/2} \sin^3 x \cos x,dx$.

Пусть $t=\sin x$, тогда $dt=\cos x,dx$. При $x=0$: $t=0$; при $x=\pi/2$: $t=1$. Получаем $$\int_0^1 t^3,dt = \left.\frac{t^4}{4}\right|_0^1 = \frac{1}{4}.$$ Пределы пересчитаны, возврат к $x$ не требуется.
