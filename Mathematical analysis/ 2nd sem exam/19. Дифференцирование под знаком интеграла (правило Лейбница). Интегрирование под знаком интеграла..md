# Дифференцирование под знаком интеграла (правило Лейбница). Интегрирование под знаком интеграла

## Правило Лейбница (постоянные пределы)

**Теорема.** Пусть $f(x,y)$ и $\dfrac{\partial f}{\partial y}(x,y)$ непрерывны на $[a,b]\times[c,d]$. Тогда $I(y)=\displaystyle\int_a^b f(x,y)\,dx$ дифференцируема и
$$\boxed{\;I'(y)=\int_a^b\frac{\partial f}{\partial y}(x,y)\,dx.\;}$$

*Доказательство.* Рассмотрим разностное отношение:
$$\frac{I(y+h)-I(y)}{h}=\int_a^b\frac{f(x,y+h)-f(x,y)}{h}\,dx.$$
По теореме Лагранжа $\dfrac{f(x,y+h)-f(x,y)}{h}=f_y(x,y+\theta h)$, $\theta\in(0,1)$. Функция $f_y$ непрерывна на компакте $\Rightarrow$ равномерно непрерывна, поэтому
$$f_y(x,y+\theta h)\rightrightarrows f_y(x,y)\quad(h\to0)\ \text{равномерно по } x.$$
По теореме о предельном переходе под знаком интеграла (равномерная сходимость, билет 10):
$$\lim_{h\to0}\int_a^b f_y(x,y+\theta h)\,dx=\int_a^b f_y(x,y)\,dx.\qquad\blacksquare$$

## Правило Лейбница (переменные пределы)

**Теорема.** Пусть $f,f_y$ непрерывны, а $\alpha(y),\beta(y)$ дифференцируемы. Тогда для $I(y)=\displaystyle\int_{\alpha(y)}^{\beta(y)}f(x,y)\,dx$:
$$I'(y)=\int_{\alpha(y)}^{\beta(y)}f_y(x,y)\,dx+f(\beta(y),y)\,\beta'(y)-f(\alpha(y),y)\,\alpha'(y).$$

*Доказательство.* Рассмотрим $\Phi(u,v,y)=\int_u^v f(x,y)\,dx$. По теореме Барроу $\Phi_v=f(v,y)$, $\Phi_u=-f(u,y)$, а $\Phi_y=\int_u^v f_y\,dx$ (правило Лейбница). По цепному правилу с $u=\alpha(y)$, $v=\beta(y)$:
$$I'(y)=\Phi_v\beta'+\Phi_u\alpha'+\Phi_y=f(\beta,y)\beta'-f(\alpha,y)\alpha'+\int_{\alpha}^{\beta}f_y\,dx.\qquad\blacksquare$$

## Интегрирование под знаком интеграла (теорема Фубини для непрерывных)

**Теорема.** Если $f(x,y)$ непрерывна на $[a,b]\times[c,d]$, то
$$\int_c^d\left(\int_a^b f(x,y)\,dx\right)dy=\int_a^b\left(\int_c^d f(x,y)\,dy\right)dx.$$

*Доказательство (схема).* Обе части равны двойному интегралу $\iint_{[a,b]\times[c,d]}f\,dx\,dy$. Строго: рассмотрим $\Psi(t)=\int_c^t\!\int_a^b f\,dx\,dy-\int_a^b\!\int_c^t f\,dy\,dx$. Тогда $\Psi(c)=0$, а $\Psi'(t)=\int_a^b f(x,t)\,dx-\int_a^b f(x,t)\,dx=0$ (по теореме Барроу и правилу Лейбница), значит $\Psi\equiv0$; при $t=d$ получаем равенство. $\blacksquare$

## Применение: вычисление "трудных" интегралов

Дифференцирование по параметру — мощный приём для интегралов, не берущихся напрямую.

## Примеры

**Пример 1 (правило Лейбница).** $I(y)=\int_0^1\dfrac{x^y-1}{\ln x}\,dx$, $y>-1$. Тогда $I'(y)=\int_0^1 x^y\,dx=\dfrac{1}{y+1}$, $I(0)=0$, значит $I(y)=\ln(y+1)$.

**Пример 2 (переменные пределы).** $I(y)=\int_0^{y}\dfrac{\sin(xy)}{x}\,dx$. Тогда
$$I'(y)=\int_0^y\cos(xy)\,dx+\frac{\sin(y^2)}{y}=\frac{\sin(y^2)}{y}+\frac{\sin(y^2)}{y}=\frac{2\sin(y^2)}{y}.$$

**Пример 3 (интегрирование по параметру).** $\int_0^1\dfrac{x^b-x^a}{\ln x}\,dx$ ($a,b>-1$). Заметим $\dfrac{x^b-x^a}{\ln x}=\int_a^b x^y\,dy$. Меняя порядок:
$$\int_0^1\!\int_a^b x^y\,dy\,dx=\int_a^b\!\int_0^1 x^y\,dx\,dy=\int_a^b\frac{dy}{y+1}=\ln\frac{b+1}{a+1}.$$
