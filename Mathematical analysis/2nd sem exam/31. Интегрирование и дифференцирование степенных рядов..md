# Интегрирование и дифференцирование степенных рядов

## Сохранение радиуса сходимости при дифференцировании

**Лемма.** Степенные ряды $\sum a_n x^n$, $\sum n a_n x^{n-1}$ (производный) и $\sum\dfrac{a_n}{n+1}x^{n+1}$ (интегральный) имеют **один и тот же радиус сходимости** $R$.

*Доказательство.* По формуле Коши–Адамара радиус определяется через $\varlimsup\sqrt[n]{|a_n|}$. Для производного ряда коэффициенты $n a_n$, и
$$\varlimsup_{n\to\infty}\sqrt[n]{n|a_n|}=\varlimsup\big(\sqrt[n]{n}\cdot\sqrt[n]{|a_n|}\big)=1\cdot\varlimsup\sqrt[n]{|a_n|}=\frac1R,$$
так как $\sqrt[n]{n}\to1$. Значит радиус тот же. Для интегрального — аналогично ($\sqrt[n]{1/(n+1)}\to1$). $\blacksquare$

## Дифференцирование степенного ряда

**Теорема.** Пусть $f(x)=\sum_{n=0}^\infty a_n x^n$ при $|x|<R$. Тогда $f$ дифференцируема при $|x|<R$ и
$$f'(x)=\sum_{n=1}^\infty n a_n x^{n-1}.$$

*Доказательство.* По теореме Абеля на любом $[-q,q]$, $q<R$, исходный ряд и ряд производных сходятся равномерно. По теореме о почленном дифференцировании (билет 21): $f'(x)=\sum n a_n x^{n-1}$ на $[-q,q]$, а значит и всюду внутри $(-R,R)$. $\blacksquare$

**Следствие.** Сумма степенного ряда **бесконечно дифференцируема** внутри круга сходимости, причём
$$f^{(k)}(x)=\sum_{n=k}^\infty n(n-1)\cdots(n-k+1)\,a_n x^{n-k},\qquad a_k=\frac{f^{(k)}(0)}{k!}.$$

## Интегрирование степенного ряда

**Теорема.** Пусть $f(x)=\sum a_n x^n$ при $|x|<R$. Тогда для $|x|<R$:
$$\int_0^x f(t)\,dt=\sum_{n=0}^\infty\frac{a_n}{n+1}x^{n+1}.$$

*Доказательство.* На $[-q,q]$, $q<R$, ряд сходится равномерно, члены непрерывны. По теореме о почленном интегрировании (билет 21):
$$\int_0^x f(t)\,dt=\sum_{n=0}^\infty a_n\int_0^x t^n\,dt=\sum_{n=0}^\infty\frac{a_n}{n+1}x^{n+1}.\qquad\blacksquare$$

> **Вывод:** степенные ряды можно дифференцировать и интегрировать **почленно** сколько угодно раз внутри круга сходимости — радиус при этом не меняется.

## Применение: суммирование рядов и разложения

Почленное интегрирование/дифференцирование позволяет находить суммы рядов и разложения новых функций из известных.

## Примеры

**Пример 1 (логарифм).** Из $\dfrac{1}{1+x}=\sum_{n=0}^\infty(-1)^n x^n$ ($|x|<1$) интегрированием:
$$\ln(1+x)=\sum_{n=0}^\infty\frac{(-1)^n}{n+1}x^{n+1}=x-\frac{x^2}{2}+\frac{x^3}{3}-\dots.$$

**Пример 2 (арктангенс).** Из $\dfrac{1}{1+x^2}=\sum(-1)^n x^{2n}$ интегрированием:
$$\arctan x=\sum_{n=0}^\infty\frac{(-1)^n}{2n+1}x^{2n+1}=x-\frac{x^3}{3}+\frac{x^5}{5}-\dots.$$
При $x=1$ (где ряд сходится по Лейбницу): $\dfrac\pi4=1-\tfrac13+\tfrac15-\dots$.

**Пример 3 (дифференцирование для суммирования).** Найдём $\sum_{n=1}^\infty n x^n$. Из $\sum x^n=\frac{1}{1-x}$ дифференцируем: $\sum n x^{n-1}=\frac{1}{(1-x)^2}$, умножаем на $x$:
$$\sum_{n=1}^\infty n x^n=\frac{x}{(1-x)^2},\quad |x|<1.$$

**Пример 4 (числовая сумма).** При $x=\tfrac12$: $\sum_{n=1}^\infty\dfrac{n}{2^n}=\dfrac{1/2}{(1/2)^2}=2$.
