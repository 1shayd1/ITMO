# Приложения определённых интегралов. Площадь поверхности тела вращения

## Постановка

Пусть кривая $y=f(x)\ge0$, $f\in C^1[a,b]$, вращается вокруг оси $Ox$. Хотим найти площадь полученной поверхности вращения.

## Идея: площадь боковой поверхности усечённого конуса

Элемент дуги длины $\Delta s$ при вращении на радиусах $y_1,y_2$ заметает боковую поверхность усечённого конуса. Её площадь:
$$\Delta P=\pi(y_1+y_2)\,\Delta s=2\pi\cdot\frac{y_1+y_2}{2}\cdot\Delta s,$$
то есть длина образующей, умноженная на длину окружности по среднему радиусу.

## Формула площади поверхности вращения

**Теорема.** Площадь поверхности, образованной вращением кривой $y=f(x)$ ($f\ge0$, $f\in C^1$) вокруг оси $Ox$, равна
$$\boxed{\;P=2\pi\int_a^b f(x)\sqrt{1+(f'(x))^2}\,dx=2\pi\int_a^b y\,ds.\;}$$

*Доказательство.* Для разбиения $a=x_0<\dots<x_n=b$ заменяем дугу на хорды; вращение хорды даёт усечённый конус с площадью
$$\Delta P_i=\pi\big(f(x_{i-1})+f(x_i)\big)\sqrt{(\Delta x_i)^2+(\Delta y_i)^2}.$$
По теореме Лагранжа $\Delta y_i=f'(\tau_i)\Delta x_i$, поэтому $\sqrt{(\Delta x_i)^2+(\Delta y_i)^2}=\sqrt{1+f'(\tau_i)^2}\,\Delta x_i$. Сумма:
$$P_T=\sum_i\pi\big(f(x_{i-1})+f(x_i)\big)\sqrt{1+f'(\tau_i)^2}\,\Delta x_i.$$
Поскольку $f$ непрерывна, $f(x_{i-1})+f(x_i)\to 2f(\tau_i)$ с точностью до $\omega_f(\lambda)$ (модуль непрерывности), и сумма стремится к интегральной сумме функции $2\pi f(x)\sqrt{1+f'^2}$, которая непрерывна. Значит при $\lambda\to0$:
$$P_T\to 2\pi\int_a^b f(x)\sqrt{1+(f'(x))^2}\,dx.\qquad\blacksquare$$

## Другие формы записи

**Параметрически** $x=x(t),\,y=y(t)$, $t\in[\alpha,\beta]$, $y\ge0$:
$$P=2\pi\int_\alpha^\beta y(t)\sqrt{x'(t)^2+y'(t)^2}\,dt.$$

**Полярно** $r=r(\varphi)$ (вращение вокруг полярной оси, $y=r\sin\varphi$):
$$P=2\pi\int_\alpha^\beta r\sin\varphi\,\sqrt{r^2+r'^2}\,d\varphi.$$

**Вращение вокруг оси $Oy$:** $P=2\pi\int_a^b x\sqrt{1+f'^2}\,dx$.

## Примеры

**Пример 1 (площадь сферы).** $y=\sqrt{R^2-x^2}$, $x\in[-R,R]$. Тогда $y'=\dfrac{-x}{\sqrt{R^2-x^2}}$, $1+y'^2=\dfrac{R^2}{R^2-x^2}$, $y\sqrt{1+y'^2}=R$:
$$P=2\pi\int_{-R}^{R}R\,dx=2\pi R\cdot 2R=4\pi R^2.$$

**Пример 2 (боковая поверхность конуса).** $y=\dfrac{R}{H}x$, $x\in[0,H]$. $1+y'^2=1+\dfrac{R^2}{H^2}$:
$$P=2\pi\int_0^H\frac{R}{H}x\sqrt{1+\tfrac{R^2}{H^2}}\,dx=2\pi\frac{R}{H}\cdot\frac{H^2}{2}\cdot\frac{\sqrt{H^2+R^2}}{H}=\pi R\sqrt{H^2+R^2}=\pi R L,$$
где $L$ — образующая. Совпадает со школьной формулой.

**Пример 3 (катеноид).** $y=\cosh x$, $x\in[0,a]$. $1+y'^2=\cosh^2 x$:
$$P=2\pi\int_0^a\cosh x\cdot\cosh x\,dx=2\pi\int_0^a\frac{1+\cosh 2x}{2}\,dx=\pi\left(a+\frac{\sinh 2a}{2}\right).$$
