# Ряды Фурье в евклидовом пространстве. Единственность. Наилучшее приближение. Неравенство Бесселя. Равенство Парсеваля. Теорема Рисса–Фишера

Рассмотрим евклидово (со скалярным произведением) пространство функций, интегрируемых с квадратом, с ОНС $\{\varphi_n\}$. Коэффициенты Фурье $c_n=\langle f,\varphi_n\rangle$.

## Ряд Фурье наилучшим образом приближает функцию

**Теорема.** Среди всех линейных комбинаций $T_N=\sum_{n=1}^N\gamma_n\varphi_n$ наименьшее среднеквадратичное уклонение $\|f-T_N\|$ достигается при $\gamma_n=c_n$ (коэффициенты Фурье).

*Доказательство.* Раскроем квадрат нормы:
$$\|f-T_N\|^2=\Big\langle f-\sum\gamma_n\varphi_n,\ f-\sum\gamma_m\varphi_m\Big\rangle=\|f\|^2-2\sum_n\gamma_n\langle f,\varphi_n\rangle+\sum_n\gamma_n^2,$$
где использована ортонормированность ($\langle\varphi_n,\varphi_m\rangle=\delta_{nm}$). Подставляя $c_n=\langle f,\varphi_n\rangle$:
$$\|f-T_N\|^2=\|f\|^2-2\sum_n\gamma_n c_n+\sum_n\gamma_n^2=\|f\|^2-\sum_n c_n^2+\sum_n(\gamma_n-c_n)^2.$$
Это минимально, когда все $(\gamma_n-c_n)^2=0$, т.е. $\gamma_n=c_n$. $\blacksquare$

## Неравенство Бесселя

**Теорема.** Для любой $f$ с конечной нормой:
$$\boxed{\;\sum_{n=1}^\infty c_n^2\le\|f\|^2.\;}$$

*Доказательство.* При $\gamma_n=c_n$ из предыдущего равенства:
$$0\le\|f-S_N\|^2=\|f\|^2-\sum_{n=1}^N c_n^2,\quad\Rightarrow\quad\sum_{n=1}^N c_n^2\le\|f\|^2.$$
Неравенство верно для всех $N$, частичные суммы ограничены $\Rightarrow$ ряд $\sum c_n^2$ сходится и $\sum c_n^2\le\|f\|^2$. $\blacksquare$

**Следствие.** Коэффициенты Фурье стремятся к нулю: $c_n\to0$ (необходимое условие сходимости ряда $\sum c_n^2$).

## Равенство Парсеваля (уравнение замкнутости)

**Определение.** ОНС называется **замкнутой** (полной), если для любой $f$ верно равенство:
$$\boxed{\;\sum_{n=1}^\infty c_n^2=\|f\|^2.\;}$$

**Теорема.** Равенство Парсеваля выполнено $\iff$ ряд Фурье сходится к $f$ в среднеквадратичном: $\|f-S_N\|\to0$.

*Доказательство.* Из $\|f-S_N\|^2=\|f\|^2-\sum_{n=1}^N c_n^2$ следует:
$$\|f-S_N\|\to0\iff\sum_{n=1}^N c_n^2\to\|f\|^2.\qquad\blacksquare$$

## Единственность разложения

**Теорема.** Если ОНС замкнута и две функции $f,g$ имеют одинаковые коэффициенты Фурье, то $\|f-g\|=0$ (совпадают почти всюду).

*Доказательство.* У $h=f-g$ все коэффициенты Фурье нулевые. По равенству Парсеваля $\|h\|^2=\sum 0=0$. $\blacksquare$

## Теорема Рисса–Фишера

**Теорема (Рисс–Фишер).** Пусть $\{\varphi_n\}$ — ОНС в полном пространстве $L^2$. Для любой числовой последовательности $\{c_n\}$ с $\sum c_n^2<\infty$ существует функция $f\in L^2$, для которой $c_n$ суть её коэффициенты Фурье, причём $\|f\|^2=\sum c_n^2$.

*Доказательство (идея).* Рассмотрим частичные суммы $S_N=\sum_{n=1}^N c_n\varphi_n$. Для $M>N$:
$$\|S_M-S_N\|^2=\sum_{n=N+1}^M c_n^2\to0\quad(N\to\infty),$$
так как $\sum c_n^2$ сходится. Значит $\{S_N\}$ фундаментальна; по **полноте** $L^2$ существует предел $f=\lim S_N$. Коэффициенты Фурье $f$: $\langle f,\varphi_k\rangle=\lim_N\langle S_N,\varphi_k\rangle=c_k$. $\blacksquare$

## Примеры

**Пример 1 (Парсеваль для $f=x$).** $b_n=\frac{2(-1)^{n+1}}{n}$, $\|x\|^2=\int_{-\pi}^\pi x^2 dx=\frac{2\pi^3}{3}$. Парсеваль (с учётом норм $\pi$): $\pi\sum b_n^2=\frac{2\pi^3}{3}$, т.е. $4\pi\sum\frac1{n^2}=\frac{2\pi^3}{3}$, откуда $\sum\dfrac1{n^2}=\dfrac{\pi^2}{6}$.

**Пример 2 (Бессель как оценка).** Для любой $f\in L^2[-\pi,\pi]$: $\frac{a_0^2}{2}+\sum(a_n^2+b_n^2)\le\frac1\pi\int_{-\pi}^\pi f^2$.

**Пример 3 (Парсеваль для $x^2$).** $a_0=\frac{2\pi^2}{3}$, $a_n=\frac{4(-1)^n}{n^2}$ даёт $\sum\dfrac{1}{n^4}=\dfrac{\pi^4}{90}$.
